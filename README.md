# Portfolio

Static (no-build) portfolio site with animations, transitions, and a command palette.

## Run locally

From the repo root:

```bash
cd portfolio
python3 -m http.server 5173
```

Open `http://localhost:5173`.

## Customize

- Edit profile data in `profile.js`
- Optional: add a resume PDF at `assets/resume.pdf` and link it from the header/hero
